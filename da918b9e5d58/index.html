win电脑之v2rayN篇

发表于 2021-02-01

介绍

今天海绵宝宝带来的是 win 电脑的一款翻墙软件，叫 v2rayN 国外软件安全有保障，放心使用，接下来教大家使用方法！

缺点呢:

链接以后没同行电脑端 clash 快

优点呢:

简单易上手。

下载软件

蓝奏云下载（可能会失效，失效发邮件hk8098801@gmail.com找我更新即可）

··········点击下载··········|密码:qqqq

然后长这样，然后我们右键他，进行解压（如果没有解压软件的去百度 360 解压下载一个安装即可！）


然后我们双击 V2rayN.exe 打开。
如果不能打开 提示无法使用 x64 或者 x86，那么你这台电脑可能不能安装翻墙软件。换台配置新一点的把，xin7 也行，只要不是 x32 就好，至于怎么看 x32.x64.x86，我们找到我的电脑—>右键—
–>属性—->找到系统类型。多少位就是 x 多少。

然后我们可以在电脑的右下角找到这个软件，双击打开。（如果在右下角找不到，重新试试看打开，然后到电脑右下角展开所有图标，可能被隐藏了）

点击订阅，选择订阅设置。

点添加。

名称随便写一个；链接我们写 v2ray 订阅链接即可。不知道 v2ray 订阅链接怎么拿的参考
这里拿天上云机场来做演示，



··········点击查看 V2ray 链接复制方法··········



完事了点确定就好了

我们在次点击软件订阅，选更新订阅。

等待一会，我们订阅成功的话，中间会显示节点信息。
软件下面会显示日志，也有显示订阅成功的字样。

我们需要链接一个节点的话，就左键选中该节点，然后右键，点击设为活动服务器。

我们选中并设置为活动服务器以后，软件下面日志也会如图一样出来一点东西。

在计算机右下角找到软件—>右键—>选 http 代理—->选开启 PAC（也就是第三个）


❀
如何查软件有没有开启 vpn
如图我们在右下角找到软件看是这个样子的就是没启动

如图我们在右下角找到软件看是这个样子的就是启动
